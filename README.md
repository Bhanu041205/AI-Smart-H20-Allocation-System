Planned Preprocessing Pipeline:

Raw CSVs
  │
  ├─► 1. Handle missing values    (mean imputation / forward fill)
  ├─► 2. Remove outliers          (IQR method)
  ├─► 3. Normalize features       (Min-Max Scaling)
  ├─► 4. Time-series alignment    (sync all datasets by date + region)
  └─► 5. Feature Engineering
              ├── Average monthly rainfall
              ├── Per-capita water consumption
              ├── Reservoir utilization percentage
              └── Seasonal indicators (monsoon / summer / winter)

🔍 Research Gaps

After surveying the above literature, 5 critical gaps were identified:

  GAP 1  ▸  Limited dataset integration
             Most studies use only one source — no combined
             weather + reservoir + population + consumption pipeline.

  GAP 2  ▸  No real-time decision support
             Systems produce predictions but stop short of
             giving actionable allocation recommendations.

  GAP 3  ▸  Static allocation strategies
             Fixed priority rules that cannot adapt to
             predicted future demand or seasonal changes.

  GAP 4  ▸  No web-based accessible platform
             No interactive interface for administrators
             to view forecasts and act on plans.

  GAP 5  ▸  Sector prioritization underexplored
             Dynamic allocation across domestic, agricultural,
             and industrial sectors is rarely optimized together.

         THIS PROJECT ADDRESSES ALL 5 GAPS ✅
